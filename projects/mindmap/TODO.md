# 마인드맵 추가 개발 목록

## 2순위 (생산성)

### PNG 이미지 내보내기
- `html2canvas` 라이브러리로 캔버스를 캡처해 PNG 다운로드
- 상단 `⬇ PNG` 버튼 추가
- 현재 viewport 배율과 무관하게 원본 크기로 출력

### 자동 레이아웃 정렬
- "정렬" 버튼 클릭 시 관계 구조를 기반으로 노드 자동 배치
- 트리 레이아웃 (루트 → 자식 방향) 우선 구현
- 관계가 없는 노드는 하단에 격자 배치

### 관계 라벨 편집
- DB `relation.label` 컬럼이 이미 존재하나 UI 미지원
- 화살표 더블클릭 또는 우측 패널 관계 항목에 라벨 입력란 추가
- `PUT /api/relations/:id` 백엔드 라우트 추가 필요

### 노드 검색/필터
- 좌측 패널 상단에 검색 input 추가
- 입력 시 일치하는 노드 하이라이트 + 캔버스 자동 스크롤(팬)
- 검색어 지우면 전체 복원

## 3순위 (고급)

### 보드 복제
- 현재 보드를 복사해 새 마인드맵으로 생성
- `POST /api/boards/:id/duplicate` 백엔드 라우트 필요 (객체·관계 일괄 복사)

### 보드 삭제
- 보드 선택 드롭다운 옆에 삭제 버튼 추가
- 백엔드 `DELETE /api/boards/:id` 라우트는 이미 존재
- 삭제 후 남은 보드 중 첫 번째로 자동 전환 (없으면 새 보드 생성)

### 터치 / 모바일 지원
- 현재 `mousedown` 이벤트만 사용
- `touchstart / touchmove / touchend` 이벤트 추가
- 핀치 줌 지원 (`Touch.targetTouches` 두 손가락 거리)

### 노드 이모지 / 아이콘
- 노드에 이모지 접두어 설정 기능 (이름 앞에 붙여 저장)
- 우측 패널에 이모지 피커 또는 빠른 선택 팔레트 추가
