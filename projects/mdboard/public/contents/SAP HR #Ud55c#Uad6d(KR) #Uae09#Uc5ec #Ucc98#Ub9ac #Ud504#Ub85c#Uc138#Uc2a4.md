# SAP HR 한국(KR) 급여 처리 프로세스

> **Molga: KR** | SAP HCM Payroll for Korea | 기준: SAP ECC 6.0 / S/4HANA

---

## 1. 개요

SAP 한국 급여(Payroll KR)는 국내 노동법, 4대 보험 법령, 소득세법에 따른 급여 계산을 자동화합니다.

| 항목 | 내용 |
|------|------|
| Country Grouping (Molga) | KR |
| 급여 기간 | 월별 (Monthly) |
| 기본 통화 | KRW (원) |
| 주요 법령 | 근로기준법, 소득세법, 4대 보험법 |
| 핵심 트랜잭션 | `PC00_MKR_CALC`, `PC00_MKR_CDTA`, `PC00_MKR_CIPE` |

## 4. 한국 급여 계산 구조

### 4.1 총지급액 (Gross Pay) 구성

```
총지급액 (Gross Pay)
├── 기본급 (IT0008)
├── 직책수당
├── 식대 (비과세 한도: 월 20만원)
├── 차량유지비 (비과세 한도: 월 20만원)
├── 연장근로수당 (통상임금 × 1.5)
├── 야간근로수당 (통상임금 × 0.5 가산)
├── 휴일근로수당 (통상임금 × 1.5 또는 2.0)
├── 연차수당 (미사용 연차 보상)
└── 기타 수당 (성과급, 상여금 등)
```

### 4.2 공제 항목 (Deductions) 구성

```
공제 합계
├── [4대 보험] 국민연금 (기준소득월액 × 4.5%)
├── [4대 보험] 건강보험 (보수월액 × 3.545%)
├── [4대 보험] 장기요양보험 (건강보험료 × 12.95%)
├── [4대 보험] 고용보험 (보수월액 × 0.9%)
├── [세금] 근로소득세 (간이세액표 적용)
└── [세금] 지방소득세 (근로소득세 × 10%)
```

### 4.3 실수령액 (Net Pay)

```
실수령액 = 총지급액 - 공제합계
```

## 6. 근로소득세 처리

### 6.1 간이세액표 적용

- 매월 급여 지급 시 **국세청 근로소득 간이세액표** 기준으로 원천징수
- 공제 대상 가족 수(IT0021)에 따라 세액 차등 적용
- SAP 내 세액표는 **법 개정 시마다 커스터마이징 테이블 업데이트** 필요

```
[간이세액 계산 흐름]
과세 급여 (총지급 - 비과세 항목)
       ↓
 간이세액표 조회 (월 급여 구간 × 부양가족 수)
       ↓
 근로소득세 확정
       ↓
 지방소득세 = 근로소득세 × 10%
```

### 6.2 비과세 항목

| 항목 | 비과세 한도 | 관련 법령 |
|------|------------|-----------|
| 식대 | 월 200,000원 | 소득세법 §12 |
| 차량유지비 | 월 200,000원 | 소득세법 §12 |
| 육아휴직급여 | 전액 | 고용보험법 |
| 출산/보육수당 | 월 200,000원 | 소득세법 §12 |
| 연구보조비 | 월 200,000원 | 소득세법 §12 |
| 야간근로수당 | 연 2,400,000원 | 생산직 근로자 한정 |

### 6.3 관련 SAP Wage Type

| Wage Type | 설명 |

| /KR21 | 비과세 급여 합계 |

## 7. 연말정산 (Year-End Tax Adjustment)

### 7.1 처리 시기 및 개요

- 매년 **2월 급여** 시 전년도 귀속 소득에 대해 연말정산 실시
- 1월 중 직원 소득공제 신고서 수집 후 SAP 입력
- 결과: 추가 납부 또는 환급

### 7.2 SAP 연말정산 처리 흐름

```
[1] 직원 소득공제 신고서 수집 (1월)
       ↓
[2] IT0161 / IT0021 데이터 업데이트
       ↓
[3] 연말정산 급여 RUN 실행 (PC00_MKR_CALC)
       ↓
[4] 연말정산 결과 검토 (추가납부 / 환급 확인)
       ↓
[5] 2월 급여에 차감 또는 환급 반영
       ↓
[6] 근로소득 원천징수영수증 발행 (PC00_MKR_YEA)
       ↓
[7] 국세청 지급명세서 전자 신고 (홈택스 연계)
```

### 7.3 주요 소득공제 항목

| 구분 | 항목 |
|------|------|
| 인적공제 | 본인, 배우자, 부양가족 1인당 150만원 |
| 특별소득공제 | 건강보험료, 고용보험료, 주택자금 |
| 세액공제 | 근로소득세액공제, 자녀세액공제 |
| 특별세액공제 | 보험료, 의료비, 교육비, 기부금 |

## 8. 퇴직금 처리

### 8.1 퇴직금 제도 구분

| 구분 | 내용 |
|------|------|
| 확정급여형 (DB) | 퇴직 시 최근 3개월 평균임금 × 근속연수 |
| 확정기여형 (DC) | 매년 연간임금총액의 1/12 이상 DC계좌 적립 |
| IRP (개인형 퇴직연금) | DC형 또는 이직 시 IRP 계좌로 이전 |

### 8.2 퇴직금 계산 (DB형 기준)

```
퇴직금 = 1일 평균임금 × 30일 × (근속일수 / 365)

1일 평균임금 = 퇴직 전 3개월 총 임금 / 퇴직 전 3개월 총 일수
```

### 8.3 퇴직소득세

- 퇴직금 지급 시 **퇴직소득세** 원천징수
- 근속연수 공제, 환산급여 공제 적용
- SAP 트랜잭션: `PC00_MKR_RETI`

## 9. 근태 연동 (Time → Payroll)

### 9.1 근태 유형별 급여 반영

| 근태 유형 | 처리 방식 | 관련 Wage Type |
|-----------|-----------|----------------|
| 정상근무 | 기본급 지급 | 기본 급여 WT |
| 연장근로 (OT) | 통상임금 × 1.5배 | /KR30 |
| 야간근로 | 통상임금 × 0.5 가산 | /KR31 |
| 휴일근로 | 통상임금 × 1.5~2.0배 | /KR32 |
| 결근 | 해당 일수만큼 일할 공제 | /KR40 |
| 연차 미사용 | 연차수당 지급 | /KR41 |
| 조기퇴직 / 중도입사 | 일할 계산 적용 | 자동 처리 |

### 9.2 통상임금 계산

```
통상임금 = (기본급 + 고정수당) / 소정근로시간

월 통상임금 시간급 = 통상임금 / (월 소정근로시간 + 주휴시간)
※ 주 40시간 기준: 월 209시간 적용
```

## 10. 급여 명세서 (Remuneration Statement)

- **트랜잭션**: `PC00_MKR_CEDT`
- 한국어 급여 명세서 양식 제공
- 근로기준법 제48조: 임금 명세서 교부 의무
- ESS(Employee Self-Service)를 통한 온라인 조회 가능

**급여 명세서 필수 기재 항목 (근로기준법 시행령):**

- 성명, 생년월일, 사원번호
- 임금 지급일 및 산정 기간
- 임금 항목별 금액 (기본급, 각종 수당)
- 공제 항목별 금액 (4대 보험, 세금)
- 실수령액

## 13. 월별 급여 처리 체크리스트

### ✅ 급여 RUN 전 (매월 말)

- [ ] 신규 입사자 마스터 데이터 등록 완료 (PA40)
- [ ] 퇴직자 처리 완료 (퇴직일, 퇴직금 설정)
- [ ] 기본급 변경자 IT0008 업데이트
- [ ] 비정기 수당 IT0015 입력 (성과급 등)
- [ ] 근태 데이터 최종 확정 및 집계 (PT60)
- [ ] 4대 보험 요율 변경 여부 확인
- [ ] 급여 제어 레코드 오픈 (PA03 → Released)
- [ ] 급여 시뮬레이션 실행 및 오류 검토

### ✅ 급여 RUN 후

- [ ] 정식 급여 RUN 실행 (PC00_MKR_CALC)
- [ ] 급여 결과 이상 유무 확인 (PC_PAYRESULT)
- [ ] 오류 사원 수정 및 재처리
- [ ] 급여 명세서 생성 및 배포 (PC00_MKR_CEDT)
- [ ] 은행 이체 파일 생성 및 은행 제출 (PC00_MKR_CDTA)
- [ ] 4대 보험 EDI 신고 파일 생성
- [ ] 원천세 신고 데이터 생성 (홈택스 제출)
- [ ] FI 회계 전기 처리 (PC00_MKR_CIPE)
- [ ] 급여 기간 확정 (PA03 → Exit Payroll)

### ✅ 연간 이벤트

## 14. 소급 계산 (Retroactive Calculation) — 한국 특이사항

| 소급 발생 원인 | 처리 방법 |
|---------------|-----------|
| 기본급 소급 인상 | IT0008 소급 적용 → 차기 급여 RUN 시 자동 처리 |
| 4대 보험 요율 변경 | 커스터마이징 테이블 업데이트 → 소급 정산 |
| 연장근로 오류 수정 | 근태 데이터 수정 → 급여 재계산 |
| 연말정산 수정 | PC00_MKR_YEA 재실행 → 차액 반영 |

> **주의**: 소급 기간이 길수록 클러스터 데이터 부하 증가. 대량 소급 시 배치 분리 처리 권장.
