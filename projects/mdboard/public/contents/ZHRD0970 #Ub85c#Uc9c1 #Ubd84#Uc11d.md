# MZHR0970F01 로직 분석
## 연장 / 휴일 / 휴일연장 / 야간 계산 로직

> 핵심 폼: `CALCULATION_NIGHT_WORK_TIME`  
> 참고: `Z_HR_PCOFF_IN_EXTWORK` 펑션과 동일한 로직 (주석 명시)

---

## 1. 연장 vs 휴일 구분 로직

### 핵심 분기 조건

```abap
IF ls_zhrt0974 IS INITIAL AND ( is_holiday = 'X' OR temp_day = '6' OR temp_day = '7' )
```

| 조건 | 처리 구분 | 저장 필드 | 할증 |
|---|---|---|---|
| 공휴일/토일 **+** 대체휴무 미신청 | 휴일근무 / 휴일연장 | `calhw`, `calhot` | 150% / 200% |
| 평일 **또는** 대체휴무 신청 | 연장근무(평일) | `calot` | 150% |

### 휴일 상세 계산

```abap
IF lt_zhrt0970-pcctm > '480'.       "하루 최대 480분(8H)
  lt_zhrt0970-calhw  = '480'.
  lt_zhrt0970-calhot = lt_zhrt0970-pcctm - '480'.
  CLEAR lt_zhrt0970-calot.
ELSE.
  lt_zhrt0970-calhw = lt_zhrt0970-pcctm.
  CLEAR : lt_zhrt0970-calot, lt_zhrt0970-calhot.
ENDIF.
```

- `pcctm` ≤ 480분 → 전부 **휴일근무(calhw)**, 휴일연장 없음
- `pcctm` > 480분 → 480분은 **휴일근무(calhw)**, 초과분은 **휴일연장(calhot)**

> **포인트**: 대체휴무(`ZHRT0974`)를 신청했다면, 토/일/공휴일이어도 **평일 연장**으로 처리됩니다.

## 2. 야간근무 계산 (bnight: 기본야간)

야간 계산은 **두 단계 필터**를 통과해야 합니다.

### 2-1. 1단계 — 즉시 제외 조건 (bnight = 0)

```abap
IF lt_zhrt0970-bsttm BETWEEN '060000' AND '130000' OR
  ( ls_zhrt0974 IS INITIAL AND ( is_holiday = 'X' OR temp_day = '6' OR temp_day = '7' ) ).
  CLEAR lt_zhrt0970-bnight.
```

아래 두 조건 중 하나라도 해당하면 기본야간 = 0:

1. 근무시작시간(`bsttm`)이 **06:00 ~ 13:00** 사이
2. 공휴일/토일인데 **대체휴무 미신청**인 경우

### 2-2. 2단계 — 연차/반차 확인

참조 테이블: `PA2001`, `ZHRT0972` (휴가유형: `0110`=연차, `0111`=오전반차, `0112`=오후반차)

| 상황 | 처리 |
|---|---|
| 연차(`0110`) 사용 | `bnight = 0` |
| 오전반차 + 오후반차 동시 | 연차 간주 → `bnight = 0` |
| 오전반차(`0111`)만 | 근무시작 **+5시간** 뒤로 밀어서 계산 |
| 오후반차(`0112`)만 | 근무종료 **-5시간** 앞으로 당겨서 계산 |
| 해당 없음 | 정상 야간 계산 진행 |

### 2-3. 실제 야간 시간 계산 공식

```
lv_begtm = bsttm
  └─ 00:00~12:00이면 +24시간 처리 (익일 연속 계산)
lv_endtm = lv_begtm + 9시간  (기본근무 8H + 휴게 1H)

반차 조정:
  오전반차 → lv_begtm = lv_begtm + 5H
  오후반차 → lv_endtm = lv_endtm - 5H

야간 인정 구간 = max(lv_begtm, 22:00) ~ min(lv_endtm, 30:00)
                ※ 30:00 = 익일 06:00

bnight = 겹치는 분(min) 단위
```

```abap
IF lv_begtm < '220000'.
  lv_begtm = '220000'.
ENDIF.
IF lv_endtm > '300000'.
  lv_endtm = '300000'.
ENDIF.
IF lv_endtm > lv_begtm.
  lv_caltm = lv_endtm - lv_begtm.
ENDIF.
```

> **변경 이력**: 원래 야간에서 휴게시간(1시간)을 차감하는 로직이 있었으나,  
> **"야간근무는 휴게시간도 포함하여 인정"** 하기로 변경되어 현재는 차감하지 않음.  
> (관련 코드는 주석 처리됨)

## 3. 최종 야간 합계 (calnt)

```abap
lt_zhrt0970-calnt = lt_zhrt0970-night + lt_zhrt0970-bnight.
```

| 필드 | 의미 | 할증 |
|---|---|---|
| `night` | 연장근무 중 야간 구간 (연장야간) | 50% |
| `bnight` | 기본근무 중 야간 구간 (기본야간) | 50% |
| `calnt` | 두 개 합산 → 급여계산 연계 | — |

---

## 4. 계산 결과 필드 정리

| 필드 | 한국어 | 단위 | 설명 |
|---|---|---|---|
| `ovctm` | 신청시간 | 분 | 연장근무 신청 시간 |
| `pcctm` | 실적시간 | 분 | 실제 처리된 시간 |
| `calot` | 연장근무 | 분 | 평일 연장 (150%) |
| `calhw` | 휴일근무 | 분 | 휴일 8H 이내 (150%) |
| `calhot` | 휴일연장 | 분 | 휴일 8H 초과분 (200%) |
| `night` | 연장야간 | 분 | 연장 중 22:00~06:00 (50%) |
| `bnight` | 기본야간 | 분 | 기본근무 중 22:00~06:00 (50%) |
| `calnt` | 야간합계 | 분 | night + bnight |

> **단위 주의**: DB(`ZHRT0970`)에는 **분(min)** 단위 저장.  
> 화면 표시 시 `/60`으로 **시간(H)** 단위 변환.  
> 계산 로직 수정 시 단위 혼동 주의.

## 5. 계산 흐름 요약

```
ZHRT0970 조회
    │
    ▼
공휴일/토일 여부 판단 (HOLIDAY_GET + DATE_COMPUTE_DAY)
    │
    ▼
대체휴무 신청 여부 확인 (ZHRT0974)
    │
    ├─[휴일 + 대체휴무 없음]──────────────────────────────┐
    │                                                    │
    │  pcctm > 480?                                      │
    │    Y: calhw=480, calhot=pcctm-480                  │
    │    N: calhw=pcctm                                  │
    │                                                    │
    └─[평일 or 대체휴무 있음]──── calot=pcctm ────────────┤
                                                         │
    ┌────────────────────────────────────────────────────┘
    │
    ▼
야간 제외 조건? (bsttm 06:00~13:00 OR 휴일+대체없음)
    │
    ├─[YES]──── bnight = 0
    │
    └─[NO]
         │
         ▼
    연차/반차 확인 (PA2001 + ZHRT0972)
         │
         ├─[연차 or 오전+오후반차]──── bnight = 0
         │
         └─[해당없음]
              │
              ▼
         22:00~30:00 구간 겹치는 분 → bnight
              │
              ▼
         calnt = night + bnight
```
